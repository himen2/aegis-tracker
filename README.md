**Aegis Tracker** — это современная, легковесная и сверхбыстрая библиотека для логирования метрик машинного обучения. В отличие от других решений, ядро Aegis написано на **Rust 🦀**, что позволяет отправлять метрики и собирать статистику системы **с нулевым влиянием (zero-overhead)** на процесс обучения вашей нейросети.

## ✨ Ключевые особенности

- 🚀 **Rust-Powered Core**: Системные метрики и фоновая отправка данных обрабатываются в изолированных потоках Rust, полностью обходя Python GIL. Ваша модель обучается на максимальной скорости.
- 💾 **Local-First Architecture**: Даже если у вас пропадет интернет или сервер недоступен, Aegis сохранит все метрики в локальную базу данных SQLite и автоматически дошлет их, когда соединение восстановится.
- 📊 **Автоматический сбор системных метрик**: Умный мониторинг CPU, RAM без тяжелых Python-зависимостей вроде psutil.
- 🔌 **Plug & Play Интеграции**: Нативные коллбеки для Keras и PyTorch Lightning (в разработке).
- 🌐 **Бесшовная работа с Jupyter**: Идеальная совместимость с Jupyter Notebook и JupyterLab.

## 📦 Установка

Библиотека распространяется в виде готовых скомпилированных бинарников (Wheels) для Windows, Linux и macOS. Установите пакет с помощью стандартного менеджера пакетов pip.

```bash
pip install aegis-tracker
```

## 🚀 Быстрый старт

Использовать Aegis так же просто, как написать встроенную функцию print. 

Сначала вам нужно импортировать библиотеку и авторизоваться с помощью вашего токена. Этот шаг сохранит токен локально, поэтому вам не придется вводить его каждый раз.

```python
import aegis

aegis.login("your_api_token_here")
```

Затем создайте новый эксперимент, указав название проекта и имя текущего запуска.

```python
run = aegis.init(project="my_cool_ai", name="ResNet_Training")
```

Теперь вы можете логировать любые метрики внутри вашего цикла обучения. Отправка происходит мгновенно, так как все сетевые запросы обрабатываются фоновым потоком на Rust.

```python
for epoch in range(10):
    loss = train_epoch()
    accuracy = validate()
    
    run.log({
        "loss": loss,
        "accuracy": accuracy,
        "epoch": epoch
    })
```

## 🧠 Как работает Rust-ядро?

Когда вы вызываете логирование метрик, Aegis не блокирует выполнение Python для отправки HTTP-запроса. Вместо этого происходит следующее:
1. Данные мгновенно передаются в сверхбыструю очередь в памяти через crossbeam-channel.
2. Независимый системный поток на Rust читает эту очередь, упаковывает метрики в батчи и отправляет их на сервер с помощью легковесного HTTP-клиента ureq.
3. Никаких лагов сети, никаких задержек от GIL — 100% ресурсов вашего процессора и видеокарты отдаются обучению модели.

## 🛠️ Сборка из исходников (Для разработчиков)

Если вы хотите внести вклад в разработку или собрать ядро под нестандартную архитектуру, вам понадобится компилятор Rust и утилита maturin. 

Сначала установите Rust с официального сайта, а затем установите инструмент сборки через pip.

```bash
pip install maturin
```

Склонируйте репозиторий на свой компьютер и перейдите в папку проекта.

```bash
git clone https://github.com/your-username/aegis-dist.git
cd aegis-dist
```

Скомпилируйте и установите пакет в вашу виртуальную среду.

```bash
maturin develop --release
```
