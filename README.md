# Aegis Tracker

Aegis — это библиотека для трекинга метрик и мониторинга ваших ML-экспериментов.

## Установка

Вы можете установить эту библиотеку напрямую из GitHub:

```bash
pip install git+https://github.com/ВАШ_ЛОГИН/ВАШ_РЕПОЗИТОРИЙ.git
```

*(Замените ссылку на ту, где будет лежать содержимое этой папки)*

## Быстрый старт

Интеграция занимает всего 3 строчки кода:

```python
import aegis

# 1. Запускаем эксперимент (он появится в вашем дашборде)
run = aegis.init(project="my_project", name="experiment_1", config={"lr": 0.001})

# 2. Логируем метрики в цикле обучения
for epoch in range(50):
    run.log({"loss": 0.5, "accuracy": 0.85})

# 3. Завершаем (опционально, если скрипт заканчивается сам)
run.finish()
```

### Поддержка фреймворков

Для PyTorch, Keras и Sklearn есть готовые коллбеки (см. документацию в веб-версии Aegis).

> **Note:** Исходный код этой библиотеки защищен обфускацией `PyArmor` для обеспечения безопасности проприетарной логики. Для работы требуется архитектура, поддерживаемая модулем `pyarmor_runtime`.
